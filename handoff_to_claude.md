# 生産管理課 ポータルサイト プロジェクト 引き継ぎ書

## 1. プロジェクト概要
- **目的**: 会社の生産管理課向け社内ポータルサイト（リンク集）の構築。
- **要件**: シンプルで機能的、かつデザイン性（モダンでプレミアム感のあるUI）を両立。
- **ホスティング**: 完全無料の要件を満たすため、GitHub Pages（`https://ni1214.github.io/portal/`）で公開済み。

## 2. 技術スタック
- **構造**: HTML5
- **スタイリング**: Vanilla CSS（CSS Variablesを使用。TailwindやSassは不使用）
- **ロジック**: Vanilla JavaScript（フレームワーク不使用）
- **アイコン**:
  - メインボタン: Font Awesome v6.5.1 (CDN)
  - 外部ツール部: Inline SVG (Notion, Slack, Google Drive公式カラー)
- **フォント**: Google Fonts 'Noto Sans JP'
- **バージョン管理**: Git / GitHub

## 4. 主な実装済み機能・デザイン
### 4.1. UI/デザイン
- **テーマ**: ダークネイビー（`--bg-primary`, `--bg-secondary`）基調
- **スタイル**: グラスモーフィズム（半透明な背景とボーダー）を活用したカードデザイン
- **インタラクション**:
  - カードホバー時の浮き上がり＋ブランドカラー/アクセントカラーのグロー（光彩）エフェクト
  - クリック時のリップル（波紋）アニメーション
- **レスポンシブ**: デスクトップ、タブレット、モバイルに対応する柔軟なCSSグリッド設定

### 4.2. 機能
- **リアルタイム検索**: `script.js` にて、検索バー（`#search-input`）の入力値に応じてカード（`.link-card`）の表示/非表示を動的に切り替え。該当なしの場合は専用のUI（`.no-results`）を表示。
- **カテゴリ分類**:
  - 管理・報告 (青系)
  - 手配・製作 (シアン系)
  - 金物・在庫 (紫系)
  - 資材・設計 (オレンジ系)
  - その他 (ピンク系)
- **外部ツールセクション**: ページ最上部にて、Notion、Slack、Google Drive へのリンクを配置。各サービス固有のインラインSVGとCSS変数によるブランドカラーホバーを実装。
- **現在時刻表示**: ヘッダー右上に現在日時をリアルタイム表示（30秒毎に更新）。

## 5. 現在の制約と注意点
- **プレースホルダーリンク**: 現状、HTML内の `<a>` タグの `href` 属性は `#`（プレースホルダー）になっています。ユーザー側でGoogle SpreadsheetやFormなどの実際のURLに順次差し替える予定です。
- **外部ツールリンク**: 外部ツールリンクには `target="_blank"` を付与済み。
